# vue-qr-reader
  > A vue web component to scan QR codes

## Features
  - Highly customizable
  - Support continuos scanning
  - All modern browser support
  - iOS 11 and above support


## Dependencies
  - [Vue.js 2x](https://vuejs.org/)
  - [jsQR](https://github.com/cozmo/jsQR)
